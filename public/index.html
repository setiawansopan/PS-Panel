Private Repository SSH Deploy Key
Untuk repo private, tambahkan SSH key server ini ke GitHub repo sebagai Deploy Key agar git pull bisa berjalan tanpa password.
Public Key — copy ke GitHub
Langkah setup
1. Copy key di atas
2. GitHub repo → Settings → Deploy keys → Add key
3. Paste key → centang Allow write access (opsional) → Add
4. Clone repo via SSH (sekali saja):
git clone git@github.com:user/repo.git /var/www/app
5. Deploy berikutnya pakai panel seperti biasa ✓
